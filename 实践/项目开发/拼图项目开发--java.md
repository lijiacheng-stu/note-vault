#### 打乱图片的方法
- 将16张图片，每张图片对应于0~15的数字，图片的文件名也和数字相对应
- 将0~15的数随机打乱，并投影到二维数组
	- 将一位数组打乱：数组从左到右遍历，随机地与某个位置交换
	- 将一维数组投影到二维 `data[i/4][i%4]=arr[i]` 
- 加载图片，contentpane和数组都是从左到右边，从上到下遍历，所以自然是乱的
