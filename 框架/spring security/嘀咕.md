1. Apache Shiro是spring security的alternative
2. Spring Boot能够根据我们对项目添加的依赖来应用自动配置
3. 编码包括加密和哈希。其中加密是可以根据输出推导出输入的。如果加密是解密用的是同一个key，那么就是对称加密；如果加密和解密就是用的不是同一个就是非对称加密，其中用于加密的是公钥而用于解密的是私钥，所以用公钥加密的文件只有私钥才能解开，这样的话能用于发送文件保证只能正确的人才能接收，还可以验证身份。哈希的话不能根据输出推导出输入，一个指定的输入就是指定的输出，可以加盐放置破解。
4. 异步方法和同步方法的调用：异步方法就是在主函数中调用一个方法，并为这个方法的执行添加一个进程，让该方法的执行和主函数中方法的执行同时进行；同步方法的调用就是主函数中调用一个方法，只有这个方法调用返回到主函数后，主函数才能继续进行。







自定义的配置类ProjectConfig继承了WebSecurityConfigurerAdapter，重写了这个类的两个方法：
- protected void configure(HttpSecurity http) throws Exception
这个方法用来管理authorization
- protected void configure( AuthenticationManagerBuilder auth)
这个方法用来管理PasswordEncoder和UserDetailsService
