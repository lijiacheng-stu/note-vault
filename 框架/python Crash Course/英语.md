普通：
- versatile 多功能的；多才多艺的；
- proficient 熟练的；精通的
- convention 规范； （如：naming convention 命名规范）
- singular 单数 
- plural 复数
- definitely 毫无疑问地；明确地 (define 定义 -> definition 定义 -> definite 明确的)
- pizzeria 披萨店
- for the sake of 为了……起见；出于……的考虑；
- poll 对……进行民意调查；调查
- halt 停止
- crash 程序崩溃
- punctuation 标点符号
符号名：
- quotes 引号 （如：single or double quotes 单双引号）
- apostrophe 撇号`'` (注： apostrophes = single quotes 单引号 ）
- parentheses 括号
- whitespace = {spaces, tabs, and end-of-line symbols}
- hash mark 井号`#`
- fleet 舰队
- modulo operator  模运算符`%`
- newline character 换行符
专业：
- off-by-one error 差一错误（多算了一个或少算了一个）
- indent 缩进 <-> 区分 intend 打算
- loop through 遍历
