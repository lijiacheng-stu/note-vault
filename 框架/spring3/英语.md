Don’t rush into learning details before properly understanding the basics

It might be unclear where the Product instance comes from. If you’ve just begun learning Spring and find yourself in such a situation, my advice is to be aware that Spring tends to have plenty of syntaxes to hide as much code as possible. Whenever you find a syntax you don’t clearly understand in an example or article, try finding the framework specification details.
