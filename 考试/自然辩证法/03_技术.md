## 1. 技术的本质
---
- 人类
- 目的：为满足自身的需要
- 时间：在实践活动中
- 手段：根据**实践经验**或**科学原理**所创造或发明的
- 本质：**各种手段和方式方法**的总和
- 体现两个方面：技术活动、技术成果
- 具有双重属性：自然属性、社会属性

## 2.科学与技术的关系
---
### 2.1 区别

|      | 科学                        | 技术                        |
| ---- | ------------------------- | ------------------------- |
| 目的   | 认识世界                      | 改造世界                      |
| 活动程序 | 从实践到理论，重点回答‘是什么’和‘为什么’的问题 | 从理论到实践，重点回答‘做什么’和‘怎么做’的问题 |
| 成果   | 一种知识形态，如概念、原理             | 一定的物质形态，如工具               |
| 评价   | 以真理性标准为内核                 | 以功利性标准为内核                 |
| 经济效用 | 间接的、不确定的                  | 直接的、确定的                   |
| 共享？  | 承认创立者的优先权，知识财富属于全人类       | 技术成果具有专利性                 |
## 2.2联系
- 科学和技术在长期的发展过程中相互影响、相互渗透、相互依赖、相互促进
- 科学为技术的发展提供理论基础，为技术创新作了知识上的准备
- 技术的需求和进步又为科学提供新的研究课题、研究工具和探索手段


## 3. 自主创新及创新型国家建设
### 3.1 为建设创新型国家建言献策
- 深化科学技术体制改革
- 建设科学合理的制度和政策体系
- 培养造就富有创新精神的人才队伍
- 发展创新文化、培育全社会的创新精神

### 3.2 创新型国家的基本特征
- 创新投入高，RD的GDP占比高
- 科技进步贡献率高
- 自我创新能力强
- 创新产出高
