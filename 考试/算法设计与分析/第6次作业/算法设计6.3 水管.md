# 算法设计与分析 6.3 水管
## ★题目描述
现在有一个水管网络，网络上有 N 个转接点，M 条水管，每条（有向）水管有容量和其单位容量的水费，请求出从起点 1 到终点 N 的最大流量及其最小水费。

## ★输入格式
第一行包括两个整数 N,M(1≤N,M≤100)，表示点数、水管数。
接下来 M 行每行 4 个数字 a,b,c,d(1≤a,b≤N,1≤c≤100,1≤d≤100) 代表节点 a 到 b 有一条容量为 c 的水管且该水管每单位流量需要花费 d 费用。

## ★输出格式
输出最大流量及其最小费用。

## ★样例输入
```
2 1
1 2 3 4
```

## ★样例输出
```
3 12
```
