# 算法设计与分析 6.1 营地
## ★题目描述
为了使QQ人无法从出生点到达营地，请在地图上设置障碍物。游戏地图可视作 $2\times n$ 的方格图，其中：
- `.` 表示QQ人可任意行走的平地；
- `#` 表示QQ人不可通过的障碍物，玩家可通过在平地上设置障碍物，即将 `.` 变为 `#` 以阻挡QQ人前进；
- `S` 表示QQ人出生点，将有大量的QQ人在该点生成，QQ人可向上/向下/向左/向右移动，且无法移动至地图外；
- `P` 表示瞬移点，移动到 `P` 点的QQ人可被传送至任意一个 `P` 点，也可选择不传送；
- `C` 表示营地。

然而用来建造障碍物的摩拉是有限的，请返回玩家最少需要放置几个障碍物才能获得胜利。
若无论怎样放置障碍物均无法获胜，请返回 `-1`。

注意：
- 地图上可能有一个或多个出生点
- 地图上有且只有一个营地

## ★输入格式
第一行一个整数 $n$ ，表示地图的列数（$1 \le n \le 10^4$）。
接下来两行表示一个 $2\times n$ 的地图。
地图仅包含字符 `.`、`#`、`C`、`P`、`S`

## ★输出格式
返回一个整数，表示最少需要放置障碍物的个数。

## ★样例输入1
```
8
S.C.P#P.
.....#.S
```

## ★样例输出1
```
3
```

## ★样例输入2
```
13
SP#P..P#PC#.S
..#P..P####.#
```

## ★样例输出2
```
-1
```

## ★样例输入3
```
7
CP.#.P.
...S..S
```

## ★样例输出3
```
4
```
