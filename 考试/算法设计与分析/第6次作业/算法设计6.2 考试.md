### 题目描述
给定一个 $m \times n$ 的矩阵，表示教室的椅子排列。如果椅子是坏的，说明这个位置不能用，就用 `#` 表示；否则，用 `.` 表示。
考生可以看到他左、右、左上、右上这四个方向上紧邻着他的学生的试卷，但是看不到直接坐在他前面或者后面的考生的试卷。
请计算该考场可以容纳的一起进行考试的最大学生人数，并保证考生无法作弊。
学生必须坐在椅子好的座位上。

### 输入格式
第一行两个整数 $n$ 和 $m$，表示教室的椅子排列（$1 \le n,m\le 8$）；
接下来一个 $n\times m$ 的矩阵表示椅子的状况。

### 输出格式
返回一个整数，表示最大可容纳的学生人数。

### 样例输入1
```
3 6
#.##.#
.####.
#.##.#
```

### 样例输出1
```
4
```

### 样例输入2
```
5 2
.#
##
#.
##
.#
```

### 样例输出2
```
3
```
