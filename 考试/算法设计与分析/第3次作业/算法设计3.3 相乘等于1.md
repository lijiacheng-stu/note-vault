# 相乘等于1
## 问题描述
给定一个长度为$n$的整数序列$a_1, a_2, \cdots, a_n$
每次操作可以选取其中的一个数字进行加1或者减1的操作，使得最终整个序列的数相乘等于1
输出最小的操作次数

## 数据范围
$1\le n \le 10^5$
$- 10^9\le a_i \le 10^9$

## 输入描述
输入有两行
第一行为整数$n$，表示序列长度
第二行为$n$个整数，用空格隔开

## 输出描述
输出满足条件的最小的操作次数

## 示例
### 输入
```
5
-5 -3 5 3 0
```

### 输出
```
13
```

### 解释说明
上述示例中，-5进行4次加1操作得到-1，-3进行2次加1操作得到-1，5进行4次减1操作得到1，3进行2次减1操作得到1，0进行1次加1操作得到1
