### 题目描述
有$n$个人编号依次为$1..n$，按此顺序排队入栈，栈容量为$p$，意味着如果已经有$p$个人在栈中，则任何人不能入栈。每个时刻要么有一个人进栈(如果可以)，要么有一个人(在栈顶)出栈。栈为空时，不能出栈。
$n$个人中，有一个人是老板(编号是$t$)，老板在进栈后，可以用指纹打开一个开关，可以直接将栈容量$p$，增加$d$，即新容量$q=p+d$。
计算有多少个不同的出栈编号序列，输出方案数 Mod $998244353$ 。
说明：栈是一个先进后出的结构，考虑一个空的数组，有两种操作：
第一种操作，插入元素到数组末尾(右侧)；
第二种操作，如果数组非空，从右侧删除(弹出)一个元素。
注意，不可以直接删除中间的元素。

### 输入格式
仅一行， 4个整数$n,t,p,d$，$(n,t,p,d) \in [1,1000]$。

### 输出格式
仅一行，一个整数表示出栈编号序列方案数。

### 样例输入
```
3 2 1 1
```

### 样例输出
```
2
```
