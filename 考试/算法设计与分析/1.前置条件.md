题目描述：
四个长度均为 n (下标为 1,2,…,n)的数组:a,b,c,d，以及对应的两种数组操作:
- 操作1:对于所有的i∈{1,2,…, n}，同时将所有的a_i改为 a_i 异或 b_i;(a_i表示数组的第i个元素，从1开始数)
- 操作2:对于所有的i∈{1,2,…, n}，同时将所有的 a_i改为 a_(d_i) +t。注意，数组 d 是1到n 的一个排列。

第一行包含三个正整数 n,m,t ，分别表示数组长度、操作次数、操作2的参数。(1<n,m < 25,0<t< 100)。接下来四行，每行 n 个整数，分别表示数组 a,b,c,d ,数组 d 是1到 n 的一个排列。(1 <a_i,b_i< 10e4,-10e4<c_i≤ 10e4)

输出：
输出包含一个整数 s，表示能得到的(`a_1*c_1 + a_2*c_2 + … + a_n*c_n`) 最大值

### ★题目描述

你可以花费A单位的时间交换序列中两个相邻的元素，当然，你也可以选择干一半摸鱼罢工。

最后你要把序列还给杰哥，但是如果杰哥发现序列中每有一个逆序对，就会把你抓过去，训话B单位时间。

接下来一行有 N 个整数，第i个数表示序列的第个元素$ D_i $。

对于80%的数据，1 <= N,A,B <=2000。

对于100%的数据，1 <= N,A,B <=100000，序列元素非负且在int范围内。

### ★输出格式

输出你要花费的最少时间。
