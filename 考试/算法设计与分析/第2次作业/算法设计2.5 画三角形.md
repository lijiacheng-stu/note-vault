# 画三角形
## ★题目描述
编程画出规模为$N$的三角形。
规模为$1$的三角形如样例所示。
规模为$X$的三角形为三个规模为$X-1$的三角形拼接而成，如样例所示。

## ★输入格式
一个整数$N$，表示三角形的规模。
对于100%的数据，1 <= $N$ <= 10。

## ★输出格式
画出对应的三角形，注意最后一个有效字符后直接回车，不要输出多余的空格。

## ★样例输入1
```
1
```

## ★样例输出1
```
 /\ 
/__\ 
```

## ★样例输入2
```
2
```

## ★样例输出2
```
   /\   
  /__\  
 /\  /\ 
/__\/__\ 
```
