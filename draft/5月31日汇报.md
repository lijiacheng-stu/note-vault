# R-CNN
## Test-time detection
![[Pasted image 20260517160953.png]]
- 用selective search算法得到2000个region proposals
- 对region proposal进行wrap，使图片缩放到适应CNN网络需要的尺寸
- 用CNN对warped region提取一维度特征，用特定类别的SVM对特征进行打分
	- 用CNN得到的是特征图，后面怎么跟FCs的？难道后面经过逐通道的平均池化吗？不，用的是展平，现在空间维度上展，后在通道维度上
- 用greedy non-maximum suppression，对region proposals进行筛选
## Training
### (1) Supervised pre-training
- 用辅助训练集((ILSVRC2012 classification),对CNN进行预训练
- 目的：预训练骨干CNN 
### (2)Domain-specific fine-tuning
- 用 (N + 1)-way classification layer替换CNN’s ImageNet-specific 1000-way classification layer
- 任务：**N+1 类多分类**
- 数据集：用warped proposal windows作为input，**IoU≥0.5 为正样本**，其余视作背景负样本
- mini-batch：32 positive windows + 96 background windows
- 目的：微调骨干CNN
### (3) Object category classifiers
- 数据集：用warped proposal windows作为input，0.5 IoU 和0.3 IoU作为标准得到input的target
- 任务：**逐类独立二分类**
- 拆掉微调分类头，固定骨干CNN
- 目的：训练特定类别的SVM
