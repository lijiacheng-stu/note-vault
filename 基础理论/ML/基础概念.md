## Enum的理解
```python
from enum import Enum
class Color(Enum):
    RED = 1
    GREEN = 2
    def hello(self):
        print(self)
```
- Color.RED是Color类的实例
- Color完全可以等价成下面这个类，有一些限制：
	- 实例化时，只能包含    RED = 1，GREEN = 2等等
```python
class Color:
	def __init__(self, name, value):
		self.name = name
		self.value = value
	def hello(self):
		print(self)
Color.RED 等价于 Color('RED', 1)
```
