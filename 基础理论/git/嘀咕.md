- 触发-f才能删除的条件是什么？
答：This is a safety feature to prevent accidental removal of data that hasn’t yet been recorded in a snapshot and that can’t be recovered from Git.
在执行指令`git rm filename` 时，才会出现，且filename在 changes not staged to be committed 和  changed to be committed 中出现时

- git rm的file-glob pattern中`/*~，` 表示以`~`结尾的文件，包括文件扩展名吗？还是扩展名以前的部分？
答：包括扩展名
