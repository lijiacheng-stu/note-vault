知识：
- 特征提取的作用：
	- This greatly reduces the variability within each digit class, because the location and scale of all the digits are now the same, which makes it much easier for a subsequent pattern recognition algorithm to distinguish between the different classes
	- Pre-processing might also be performed in order to speed up computation.
- 新的数据必须进行跟训练集相同步骤的预处理
- 特征提取的目的：the aim is to find useful features that are fast to compute, and yet that also preserve useful discriminatory information enabling faces to be distinguished from non-faces.（找到能快速计算，但是可以保留区分信息的有效特能）
- 预处理会丢失信息，如果丢失的信息对问题的解决很重要，那么可能会影响系统整体的精度
