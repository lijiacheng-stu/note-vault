Series:
 - Series是由index和values组成的。 其中values是一个一维的numpy array。
 - Series是泛化的一维numpy array
 - Series是特化的dictionary
 - Series的构造方法：
	 - 通过data + index
	 - 通过 dic（ + index的方法）

DataFrame:
- DataFrame是泛化的Numpy Array。Thus the `DataFrame` can be thought of as a generalization of a two-dimensional NumPy array, where both the rows and columns have a generalized index for accessing the data.
- DataFrame是特化的字典。 we can also think of a `DataFrame` as a specialization of a dictionary. Where a dictionary maps a key to a value, a `DataFrame` maps a column name to a `Series` of column data.
- `states['area']` 得到的是Series，而不是行。Because of this, it is probably better to think about `DataFrame`s as generalized dictionaries rather than generalized arrays

Index：
-  Index as Immutable Array ：和numpy array不一样的地方在于array的内容不可变。这样的好处是，共享index时，不需要担心因为无意识的修改而产生的潜在影响
- Index as Ordered Set ： index和index之间是可以进行集合运算的。这样的好处是 促进joins across datasets的操作, which depend on many aspects of set arithmetic.
