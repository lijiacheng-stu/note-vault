	- 45：输入形式为列表时，解析形式也会是列表
	```

	# 处理：直接解析即为list型，如下
	ls = eval(input())

	```
	- 46：输入形式为元组时，怎样转成对应数字列表？
	```
	# 输入（元组形式）：9,10,8,9,10,7,6,8,7,8
	# 处理
		# 方法一：将元组看成字符串，用‘,’划分成字符串列表，再对列表中每一项解析成对应的数字
		str_list_scores = str(input()).split(',')
		int_list_scores = [int(item) for item in str_list_scores]
		# 方法二：解析元组，再将元组转成字符串
		int_list_scores = list(eval(input()))
	```
