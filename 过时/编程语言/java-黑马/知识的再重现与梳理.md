### 方法的重写中，子类中改变父类方法中的参数列表的名称，是否能重写成功？
能！,Zi-->Fu,子类重写父类中method方法，改形参s为stri，检验通过
```java
class Fu{  
    public void method(String s){}  
}  
  
class Zi extends Fu{  
    @Override  
    public void method(String stri){  
        System.out.println("hello world");  
    }  
}
```
### 打印数组和集合的区别
打印数组，返回的是数组的地址值；打印集合，返回的是遍历每个集合中的对象的toString()方法。
