1、源代码是.java，编译后为.class，编译使用的命令为javac,运行使用命令java（运行时不加后缀名）
2、类名和文件名要相同
3、import语句后面也要加上分号
4、一个Java文件中可以有多个类，但是只有一个类能被public修饰，被public修饰的类一定得和文件名相同
5、java是强类型语言：每种数据在java中都有各自的数据类型
6、先占用的资源最后再关闭
