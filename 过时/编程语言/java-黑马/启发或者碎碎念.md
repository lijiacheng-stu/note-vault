1、对创建的对象，即可以先创建，再对对象的属性进行赋值，也可以直接使用构造器对方法进行赋值
```java
//方法一：更优雅
Person p1=new Person("老王"，13);
//方法二
Person p2=new Person();
p2.setName("老李");
p2.setAge(14);
```
