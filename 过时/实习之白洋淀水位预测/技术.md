### 2、使用MATLAB实现对excel文件的读写操作
#### 2.1写:table写入excel
- 创建table对象
```
% Step 1: 创建一个表格数据

ID = [1; 2; 3]; % ID 列，数值型

name = {'张三'; '李四'; '王五'}; % 姓名列，字符型

age = [28; 22; 35]; % 年龄列，数值型

income = [5000; 6000; 7000]; % 收入列，数值型

% 创建表格

dataTable = table(ID, name, age, income, ...

'VariableNames', {'ID', '姓名', '年龄', '收入'});
//究极创建方法，能够实现对列名字的定义
```
- 实现对excel的写
```
writetable(dataTable, filename); //向指定文件写数据
writetable(dataTable, filename, 'WriteMode', 'append'); //实现对指定文件的数据进行追加
```
